# Budget Analysis Project
This project analyzes a "6-month budget dataset of 500 individuals", focusing on spending behavior, income–expense patterns, and savings trends. The goal is to derive actionable financial insights using structured data analytics techniques.

## Project Objectives
- Clean, preprocess, and validate large-scale budget data.
- Identify spending trends, saving behaviors, and monthly variances.
- Generate meaningful insights using exploratory data analysis (EDA).
- Present findings through visualizations and summary metrics.

## Tools & Technologies Used
- **Python**
  - Pandas
  - NumPy
- **Excel**
  - Pivot tables, summary metrics, charts
- **Jupyter Notebook**
- **GitHub** for version control and documentation

## Analytical Processes
- **Data Cleaning & Preparation**
  - Handling missing values  
  - Standardizing column formats  
  - Fixing inconsistent entries  

- **Exploratory Data Analysis (EDA)**
  - Income vs. expense comparison  
  - Category-wise spending behavior  
  - Monthly variance and trends  
  - Savings distribution analysis  

- **Visualization**
  - Time-series expense trends  
  - Category segmentation charts  
  - Income–savings comparison  

## Key Insights (Example)
- Identified months with highest expense spikes and savings drop.  
- Categorized spending patterns to highlight overspending areas.  
- Highlighted segments of individuals with consistent savings growth.

## Dataset
- Includes monthly income, expenses, savings, and category data for **500 individuals across 6 months**.  
- Files Used:
  - `budget_dataset_500_persons_6months.csv`
  - `Budget planning.xlsx`
